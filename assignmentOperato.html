assignment operator is used to assign value to a variable
               = means value assign
               += eg: x+=y same as x= x+y
               -= eg: x+=y same as x= x-y
               *= eg: x+=y same as x= x*y
               /= eg: x+=y same as x= x/y
               %= eg: x+=y same as x= x%y
               **= eg: x+=y same as x= x**y
